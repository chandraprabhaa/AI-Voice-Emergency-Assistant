## ✅ Requirements

Install dependencies using:

```bash
pip install -r requirements.txt
```

### Libraries Used

- streamlit
- langchain
- groq
- python-dotenv
- audio-recorder-streamlit
- streamlit-js-eval
- openai-whisper
- torch
- gtts
- ffmpeg-python

## 🚀 Features

- 🎤 Voice + Text Emergency Detection (Whisper AI)
- 🧠 AI Emergency Detection & Classification (Groq + LangChain)
- 🔥 Emergency Severity Score (1–10 Risk Analysis)
- 🔊 AI Voice Response (TTS)
- 🌍 Multilingual Support (Auto Detect)
- 📍 Live Location Sharing
- 🏥 Nearby Hospitals / Police / Ambulance
- 💬 WhatsApp Emergency Alert
- 📞 Emergency Contact Call Link
- 🧠 Session Memory Chat
- 📄 Download Chat History

## 🔐 .env File Setup

Create a `.env` file in the root directory:

```env
GROQ_API_KEY=your_groq_api_key_here
```

👉 Replace `your_groq_api_key_here` with your actual API key

## ⚡ How to Run Project

