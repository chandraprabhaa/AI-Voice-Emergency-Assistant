## ✅ Requirements

Install dependencies using:

```bash
pip install -r requirements.txt
```

### Libraries Used

- streamlit
- langchain
- groq
- python-dotenv
- audio-recorder-streamlit
- streamlit-js-eval
- openai-whisper
- torch
- gtts
- ffmpeg-python

## 🚀 Features

- 🎤 Voice-Based Emergency Detection (Whisper AI)
- ⌨️ Text Emergency Support
- 🧠 AI Emergency Detection & Classification (LangChain + Groq)
- 🔊 AI Voice Response (Text-to-Speech)
- 🌍 Multi-Language Support (Auto Detect + Response)
- 📍 Live Location Sharing
- 🏥 Nearby Hospital Suggestions
- 🚓 Nearby Police Station Suggestions
- 🚑 Nearby Ambulance Suggestions
- 💬 WhatsApp Emergency Alert System
- 📞 One-Click Emergency Calling
- 🧠 Conversation Memory (Session Based)
- 📄 Download Emergency Chat History
